# 🎁 Interactive Eid Greeting Card

A modern and interactive Kurban Bayramı greeting card built with pure HTML, CSS and JavaScript.

## ✨ Features

- Animated gift boxes
- Particle explosion effects
- Glassmorphism UI
- Responsive mobile-friendly design
- Randomized greeting messages
- Smooth animations and transitions

## 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## 📱 Responsive Design

The project is optimized for both desktop and mobile devices.


## 💡 How to Use

1. Open the page
2. Click one of the gift boxes
3. Reveal a surprise Bayram message 🎉
